## Getting Started

### Prerequisites

- yarn

### Installation

1. Clone the repo

```sh
git clone https://gitlab.com/WizardsUniverse/web/webapi.git
```

2. Install packages

```sh
yarn
```

3. Create and setup your .env file with the .env.exemple

4. Start your local docker BDD

5. Run the API

```sh
yan start:dev
```
